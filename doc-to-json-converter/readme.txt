Convert Chapter level word document to json.
Saves the json file in "./output" folder.
Saves the images in "./output" folder.
The input-file should be in the same folder as the script.
The input file name should be in the format <chapter>.docx
Do not include topic numbers in the input file name.

Command Line Usage:
python doc_to_json.py <standard> <subject> <input-file>
Example:

python doc_to_json.py 6 science 10.docx
